# 2.凹向，拐点

>[!note]  求曲线的凹凸性与拐点的一般步骤
>$(1)找到f(x)的二阶不可导点，并求出f^{''}(x)=0的实根(找可疑点)$
>$(2)二阶不可导点、f^{''}(x)=0实根,将定义域分为若干区间，在每个区间上判断f^{''}(x)的正负，$
>$从而确定f(x)的凹凸区间$
>$(3)使得左右两侧f^{''}(x)反号的可疑点，即为f(x)的拐点$
